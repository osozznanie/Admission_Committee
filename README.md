# Admission_Committee

The Admission_Committee project is a Java Spring application that manages the admission process for an educational institution. It provides the following key functionalities:

1. **Upload Applicant's Photo during Registration**

   - Applicants can upload their photos during the registration process.
   - The uploaded photos are stored and associated with each applicant's profile.

2. **Administrator Registration of Applicant's Points**

   - Administrators have the capability to register and update points for applicants.
   - This is essential for evaluating and ranking applicants.

3. **Calculation of Points and Enrollment Determination**

   - The system calculates the total points for each applicant based on specified criteria.
   - It determines which applicants are eligible for enrollment based on predefined thresholds or rules.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Java Development Kit (JDK)
- Apache Maven
- MySQL Database (or a database of your choice)
